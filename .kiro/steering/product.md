# Product Overview

Fusen は macOS 向けの付箋アプリ。Obsidian などで管理している `.md` ファイルを、常に手前に表示されるフローティングウィンドウで閲覧・編集できる。作業フローを止めずにノートを参照するためのユーティリティ。

## Core Capabilities

- **Markdown Live Preview** — カーソルのあるブロックは raw Markdown、それ以外はレンダリング済みで表示する Obsidian ライクな編集体験
- **Always-on-top フローティングウィンドウ** — NSPanel による常時最前面表示。複数ノートを同時に開ける
- **Global Hotkey** — ノートごとにキーボードショートカットを割り当て、即座に表示/非表示を切り替え
- **外部エディタ連携** — ファイル監視により VS Code や Obsidian での編集がリアルタイムに反映
- **カスタマイズ** — カラーテーマ (6色)、透明度、フォントサイズをノートごとに設定可能

## Target Use Cases

- 開発中に TODO やメモを付箋として手元に置いておく
- Obsidian のノートを作業中に参照する
- タイル型ウィンドウマネージャ (aerospace 等) 環境で、ワークフローを崩さずにノートを表示する

## Value Proposition

plain `.md` ファイルをそのまま扱い、専用フォーマットやデータベースを持たない。既存のノート管理ワークフロー (Obsidian、dotfiles) にそのまま統合できる軽量ユーティリティ。
